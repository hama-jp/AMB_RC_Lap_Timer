# RC_AMB_Lap_Timer
AMB P3デコーダー対応の個人練習用のWebアプリ

## 実行環境
* スマートフォン、タブレットのWebブラウザ
* ページは無料のWebホスティングサービスにデプロイ（本アプリの機能を踏まえて適当なものを選定）

## 主な機能（実現したいこと）
### 設定画面
* LAN上のAMBレコーダーのIPアドレス、ポートの設定画面
* トランスポンダー番号の設定（できればローカル端末に保存したい）

### 音声読み上げ、表示
* 設定したポンダーのラップタイムを、できるだけレスポンス良くすぐに音声で読み上げ
* ポンダーごとに、ラップタイムの履歴を表示

### UI
* シンプルかつモダンなデザイン
* UIデザインは、デザイン自動生成AIを試用。DESIGN.mdで固めたい

## 実装しないこと
* ラップタイムの履歴の長期保存（MyLapsアプリなどで十分）
* ほかの人のラップタイムとか、順位の整理（ほかの人が走行していても、あくまで個人練習に特化）

## その他
* AMB P3デコーダの挙動を模擬したテスト環境
* Github ActionsでCI/CDを準備する
* 将来も機能拡張や他アプリでも流用しやすく、見通しを立てやすいコード構成を設計
